# PatientApp

## Installation Guidance 
### Install virtual environment 
``` 
    >> conda create --name venv python=3.10 
    >> conda activate venv 

```
### Install dependencies 
   
```
    >> pip install -r requirements.txt

```
### Prepare sample database
```
    >> python test_db.py

```
### Run application

```
    >> streamlit run main.py

```
